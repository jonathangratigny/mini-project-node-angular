# Mini-project Backend (Node + Express + Sequelize + SQLite)

## Setup

1. `cd backend`
2. `npm install`
3. `npm run migrate`  # creates the SQLite DB and some seed data
4. `npm start`

API endpoints:
- GET /api/tasks
- GET /api/tasks/:id
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
